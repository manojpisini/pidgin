PIDGIN Docs

The Agent Handoff Protocol.

PIDGIN standardizes how AI agents hand off work with safety, clarity, and control. Small packets. Strong guarantees. Built for production.

🛡 DESIGNED FOR RELIABILITY
Built-in safety gates (SG-1..SG-9), explicit do/deny controls, human oversight flags, and full audit logs.
Explore Safety Rules →
✨ SIMPLE & COMPOSABLE
Tiny packets, clear lifecycle, and composable workflows that fit any agent stack or runtime.
View Packet Format →
⚡ PRODUCTION READY
Rust core, async by default, metrics first, and ready for scale.
Install PIDGIN →

SAMPLE PACKET

@run <run_id>
wf=generic_review
mode=draft
do= [draft,review]
deny=[publish,send,delete]
risk=med
human=yes
user=rg_9821
out=DeepResearch
v=1.0.0
# packet is signed and timestamped

COMMANDS

COMMANDDESCRIPTIONEXAMPLE
parseParse raw packetpgn parse packet.pgn
validateValidate packet (schema, types, sign)pgn validate packet.pgn
checkRun safety rules (SG-1..SG-9)pgn check packet.pgn
resolveResolve approvals / overridespgn resolve packet.pgn
expandExpand into execution planpgn expand packet.pgn
runExecute workflowpgn run plan.json
measureCollect metrics & generate reportpgn measure --last
docsOpen documentationpgn docs
